## Setup

## Claim

For any non-empty `X ∈ C`, the sequential colimit

`colim(X → D(X) → D²(X) → ⋯) = X + ⟨α⟩`

is infinite and dense in `ℝ / ℤ`, and therefore does *not* lie in **C** (which requires finite subsets).

## Argument as stated

- `D^n(X) = X ∪ (X + α) ∪ (X + 2α) ∪ ⋯ ∪ (X + nα) = X + {0, α, 2α, …, nα}`.
- `⋃_{n} D^n(X) = X + ⟨α⟩` where `⟨α⟩ = {nα : n ∈ ℤ}` is the infinite cyclic subgroup generated by `α` in `ℝ / ℤ`.
- By **Weyl's equidistribution theorem** (Weyl 1916): if `α ∈ ℝ / ℤ` is irrational, `{nα : n ∈ ℤ}` is equidistributed and hence dense in `ℝ / ℤ`.
- A dense infinite subset of `ℝ / ℤ` is not finite, hence does not belong to **C**.

Therefore the colimit escapes **C**.

## What a validator should confirm

1. The computation `D^n(X) = X + {0, α, …, nα}` is correct.
2. The sequential colimit in **C**-as-poset-of-finite-subsets is exactly the union (since inclusions compose to unions in this poset).
3. Weyl's 1916 theorem applies as cited. A direct density argument (without full equidistribution) would also suffice for the "escape" conclusion; the paper uses the stronger equidistribution statement for rhetorical force.
4. The conclusion — the colimit does not lie in **C** — is the mathematically precise content of "the comma is irresolvable from inside **C**."

## Implication for the paper

Together with [`music-kernel-03`](music-kernel-03-terminal-coalgebra.md), this replaces the earlier D4 with two honest theorems: (a) no terminal coalgebra, (b) the natural D-closure escapes the ambient category. This is weaker than the original D4 claim but is actually true and captures the paper's core intuition.

# Resolved validation items

This file records validation items that have been resolved — verified, corrected, disputed, or withdrawn. It serves as the project's memory of what the external validation process has actually produced.

Each entry notes:
- The claim file (now closed)
- The validator(s) or corrector(s) with their relevant background
- The resolution (verified / corrected / disputed / withdrawn)
- A pointer to the paper revision that absorbed the outcome
- The date the resolution was recorded

## Template for new entries

When moving an item from [`OPEN.md`](OPEN.md) to this file, use:

```
### `<claim-slug>` — <one-line description>
- **Claim file (closed):** [`claims/<slug>.md`](claims/<slug>.md)
- **Validator(s):** <name>, <background>
- **Resolution:** verified / corrected / disputed / withdrawn
- **Paper revision absorbing outcome:** Paper N v<X.Y> § <section> (<date>)
- **Notes:** <any additional context>
- **Date recorded:** YYYY-MM-DD
```

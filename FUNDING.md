# Funding

The FalseWork Papers project operates with **no external funding** at present.

## What this means for contributors

- Validation, correction, formalization, and review contributions are made on a voluntary basis in exchange for named acknowledgement and (for graduate students) offered reference letters.
- No bounties, stipends, or cash compensation are currently available.
- If and when funding does materialize, active bounties will be announced in this file and in [`CONTRIBUTING.md`](CONTRIBUTING.md). Contributions made before funding is activated will remain in the record and will be retrospectively considered for any funded work they enabled, at the author's discretion.

## GitHub Sponsors

GitHub Sponsors is **deferred**, not planned. Enabling Sponsors on the `thefalsework` organization requires full bank, tax, and identity verification through Stripe Express even for a $0/month goal; at the project's current stage (no incoming funding, no active bounties, no concrete funders asking), that setup overhead is not justified.

If a concrete funder, foundation, or bounty offer does emerge, Sponsors can be enabled in an afternoon. Until then, please reach out directly at `chris@falsework.dev` — direct contact is the cleanest path.

## Targets that would benefit from funding

If the project were funded, the author would allocate it to:

1. **Mathematician bounties** for bounded validation work (see [`validation/OPEN.md`](validation/OPEN.md) for open claims). Typical bounded engagement: $200–$500 per one-to-two-hour specialist review.
2. **Lean formalization bounties** for substantive mathlib4-compatible proofs. Typical: $1,500–$3,000 per multi-week formalization.
3. **Translation bounties** for paper translations into other languages.
4. **Student research stipends** for graduate-level follow-on work extending the framework.

None of these is promised. This section exists to document what the author would do with funds if they appeared — not as a solicitation and not as an implicit obligation.
